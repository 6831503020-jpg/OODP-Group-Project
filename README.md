📌 Overview

A centralized system for managing university clubs, events, and student participation. It helps admins, club leaders, and students easily organize and access club activities.

🎯 Purpose
Improve communication
Reduce manual work
Provide easy access to club and event information
⚠️ Problem

Club information is scattered across multiple platforms, causing missed events and inefficient management.

⚙️ Features
User Management (roles & login)
Club & Member Management
Event Management & Registration
Announcements & Notifications
Recruitment System
Reviews & Feedback
Search & Filter
